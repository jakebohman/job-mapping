# Job Maps

**Where is hiring demand in a US city unusual?** This tool ranks major metros by
how far each one's *mix* of job postings departs from the national mix — so San
Francisco shows up heavy on tech, Miami on hospitality, Chicago on logistics. It
is an analysis tool, not a job board: you can't click through to apply.

The point is the **deviation index** — a ranked list that does the looking for
you, because a raw map of "where are the most jobs" just reproduces the
population map and teaches nothing.

the demo; **`metro_map.py` → `map.html`** is a secondary single-metro view.

## Run it

```sh
pip install -r requirements.txt
export ADZUNA_APP_ID=...  ADZUNA_APP_KEY=...   # free key: developer.adzuna.com

python pipeline/panel.py         # builds the deviation index (~25 calls/min, resumable)
cd site && python -m http.server 8000    # then open http://localhost:8000
```

`panel.py` is throttled to Adzuna's free-tier rate limit and caches each metro
as it goes, so a network blip won't lose progress — just re-run to continue.

Every module is self-testing (no network, no keys):

```sh
python pipeline/geo.py --selftest     # likewise: ingest, classify, bls, panel
```

## How the number is built

```
Adzuna posting COUNTS per sector, per metro   (a census — not a ranked sample)
        │
        ├─ sector's share of THIS metro's postings
        └─ sector's share of ALL metros' postings  (national baseline)
        │
        ▼
   deviation = log2(metro share ÷ national share)   →  ranked, top ones shown
```

Counts, not sampled-and-classified postings: Adzuna's search is ranked by a few
high-volume advertisers, so a *sample* is biased (one hospital network made
Dallas read 94% healthcare), but a *count* is not. This is the core lesson —
see PROJECT.md.

## Known limits (deliberate, documented in PROJECT.md)

- Job-posting demand, not employment; postings over-represent high-churn work.
- Adzuna's own ~30 sector categories, not O*NET occupations (the occupation
  path needs a cleaner text source than Adzuna's 500-char snippets — NLx).
- 10 metros in the demo, sentences are templated, storage is JSON — all easy to
  scale up later.
